# Yummy Garden 🌸

A cute, gamified meal-tracking app that helps you build healthy eating patterns by growing a virtual garden. Every meal you log makes your garden bloom!

## Features

- **Meal Logging** — Type what you ate or tap a quick-pick emoji (🍓🫐🍇🥭🍌 and more)
- **Growing Garden** — Each meal plants a flower or fruit in your animated garden
- **Butterflies & Rainbow** — Unlock butterflies (1 per 3 meals) and a rainbow (after 5 meals)
- **Gentle Reminders** — Soft, encouraging nudges when it's been a while since your last meal
- **Streak Tracking** — See your daily eating streak and total meals logged
- **Weekly Progress** — Visual weekly grid showing your consistency (🌑→🌱→🌷→🌸)
- **Celebrations** — Confetti and milestone celebrations keep you motivated
- **Contextual Greetings** — Time-of-day aware messages that gently encourage nourishment

## Tech

Pure HTML/CSS/JS — no build step, no dependencies. Data persists in localStorage.

## Deploy

Static site — deploy anywhere. Designed for **Cloudflare Pages**:

1. Connect this repo to Cloudflare Pages
2. No build command needed
3. Set output directory to `/` (root)
4. Done! Your garden is live 🌺

## Philosophy

Yummy Garden is designed to be warm, encouraging, and never guilt-inducing. The goal is to help users (especially women) remember to eat consistently throughout the day through light, joyful gamification. Think: rainbows, sunshine, butterflies, and flowers — not calorie counting or restriction.
